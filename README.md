# StudyQuest

CP3406 Assessment 3 education app.

StudyQuest will be developed as an educational quiz app for high school and university students. The app will support learning through quiz practice, mistake review, progress tracking, settings, and local learning statistics.

## Planned Core Screens

- Landing page
- Quiz activity screen
- Review screen
- User statistics screen
- Settings screen

## Planned Technical Features

- Kotlin and Jetpack Compose
- Material Design 3 UI
- Navigation Component
- ViewModels, repositories, and dependency injection
- Internet API integration for quiz content
- Room database for learning history and statistics
- Unit tests for scoring, data handling, and app logic
